# Building a Financial Advisor

## Description
This project demonstrates the creation of a financial advisor using AI and machine learning models. It includes a Jupyter notebook that showcases the use of OpenAI's Whisper model for transcription, processing YouTube video content, and analyzing financial decisions.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Contributing](#contributing)
- [License](#license)

## Installation
To run the notebook, you need to install the required dependencies. You can do this by running:
```bash
pip install -r requirements.txt
```

## Usage
1. Clone the repository:
```bash
git clone <repository_url>
```
2. Navigate to the project directory:
```bash
cd building_a_financial_advisor
```
3. Open the Jupyter notebook:
```bash
jupyter notebook Building_a_Financial_Advisor.ipynb
```

## Features
- **GPU Processing**: Utilizes GPU for efficient processing.
- **OpenAI Whisper Model**: Loads and uses the Whisper model for transcription.
- **YouTube API Interaction**: Downloads and processes YouTube video content.
- **Financial Analysis**: Discusses investment decisions, education value, and long-term financial planning.
- **Client-Advisor Relationship**: Emphasizes the importance of trust and clear communication in financial advisory.
